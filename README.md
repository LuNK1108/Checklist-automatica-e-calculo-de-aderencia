# Checklist Automática e Cálculo de Aderência

Sistema web desenvolvido para automatizar a verificação de não conformidades em documentos de software, realizando o cálculo de aderência de forma automática.

## Funcionalidades

- Cadastro de itens de checklist
- Verificação de conformidades e não conformidades
- Cálculo automático da porcentagem de aderência
- Exibição dos resultados da avaliação
- Apoio à análise de qualidade de documentos de software

## Tecnologias utilizadas

- JavaScript
- PHP
- HTML
- CSS

## Contexto

Projeto desenvolvido com foco na automação de checklist e análise de aderência em processos de qualidade de software.
